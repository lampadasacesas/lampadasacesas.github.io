## Sobre este site

**lampadasacesas.com.br** e um site estatico de evangelizacao catolica com legendas em portugues brasileiro.

- Hospedagem: GitHub Pages
- Tecnologia: HTML/CSS/JavaScript estatico
- Sem banco de dados, sem sistema de login, sem dados de usuarios

## Como reportar um problema de seguranca

Se voce encontrou:
- Conteudo inadequado ou malicioso no site
- Link quebrado ou redirecionamento incorreto
- Qualquer comportamento suspeito

Entre em contato por e-mail descrevendo o problema.
Responderemos em ate 72 horas.

## O que NAO e considerado vulnerabilidade

- O site incorpora videos do YouTube via embed (comportamento esperado)
- O site usa Google Analytics para medir audiencia (comportamento esperado)
